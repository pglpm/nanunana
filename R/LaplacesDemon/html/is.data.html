is.data

Description

This function provides a logical test of whether or not a given list of data meets minimum criteria to be considered data for LaplaceApproximation or LaplacesDemon.

Usage

is.data(Data)

Arguments

Data

This is a list of data. For more information, see the LaplacesDemon function.

Details

This function is useful for testing whether or not a list of data meets minimum criteria to be considered data in this package. The minimum requirements are that Data is a list, and it contains mon.names and parm.names.

This function is not extensive. For example, it does not match the length of parm.names with the length of Initial.Values, or compare the length of mon.names to the number of monitored variables output from the Model specification function. Additional checks are conducted in LaplaceApproximation and LaplacesDemon.

Value

The is.data function returns a logical value. It returns TRUE if Data meets minimum requirements to be considered data in this package, and FALSE otherwise.
